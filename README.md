# Angular Basics: Using localStorage Object To Keep Data in the Browser

## Project Summary
Learn how to persist user preferences and simple data in the browser using the `localStorage` API within an Angular application.

## How to Run
### Installation
1. `npm install` 

### Running
1. `ng serve` 
2. Change the background color preference and reload the page to see it persisted.
